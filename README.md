# Tailwind CSS Learning Project 🚀

This repository contains my first project while learning **Tailwind CSS**.
The goal of this project is to understand how Tailwind utility classes work and how they can be used to build **modern responsive websites**.

Instead of learning only from tutorials, I am studying real UI examples and experimenting with Tailwind components.

---

## 📌 What I Learned

Through this project I explored some important Tailwind concepts:

* Responsive design using `sm`, `md`, `lg`, `xl`
* Flexbox and Grid layouts
* Spacing utilities (`p-`, `m-`, `px-`, `py-`)
* Typography utilities (`text-`, `font-`)
* Colors and backgrounds
* Hover effects and transitions
* Glassmorphism UI using `backdrop-blur`
* Sticky navigation bar
* Layering elements using `z-index`
* Responsive cards and layouts
